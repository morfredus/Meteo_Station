# Contribuer

Les contributions sont les bienvenues !

1.  Forkez le dépôt.
2.  Créez une branche de fonctionnalité (`git checkout -b feature/MaSuperFonction`).
3.  Commitez vos changements.
4.  Poussez vers la branche.
5.  Ouvrez une Pull Request.
